Slide Rush

Juego de lógica basado en el puzzle deslizante clásico. Cuadrícula variable de 3×3 a 10×10. Desarrollado con Unity.

📦

Este proyecto ya no está disponible públicamente

El código fuente no es accesible actualmente. Fue desarrollado como proyecto personal enfocado en lógica de juego y diseño de mecánicas.

Sobre el proyecto

Slide Rush es un juego de lógica basado en el clásico puzzle deslizante. El jugador debe reorganizar piezas de una imagen dentro de una cuadrícula que varía desde 3×3 hasta 10×10, utilizando un sistema de movimiento con un espacio vacío.

El objetivo es reconstruir la imagen original en el menor número de movimientos posible, aumentando progresivamente la dificultad al incrementar el tamaño de la cuadrícula.

Desarrollado como proyecto personal con foco en lógica de juego, diseño de mecánicas y desarrollo mobile con Unity.

Lo que aprendí

  • Diseño de mecánicas basadas en lógica y estructuras de datos
  • Manejo de matrices 2D para representar estados de juego
  • Desarrollo de sistemas de dificultad escalable
  • Optimización de UI para dispositivos móviles con Unity
  • Importancia del testing de edge cases en lógica de puzzles

Si lo rehaciera…

  • Animaciones fluidas al deslizar piezas (tweening con DOTween)
  • Sistema de niveles con progresión guardada localmente
  • Modo contrarreloj con tabla de clasificación
  • Soporte para imágenes del carrete del usuario
  • Efectos de partículas al completar el puzzle
